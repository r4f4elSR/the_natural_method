This is a digital reproduction of the book English By The Nature Method by Arthur Jensen, but with new phonetic transcriptions, obtained with [toPhonetics](https://tophonetics.com/). In this method, you learn the language without using translations, just the language itself.

**IMPORTANT.1:** Before starting each chapter, we recommend that you familiarize yourself with the language by listening to native speakers. \
**IMPORTANT.2:** If you don't understand a word, don't use a translator, use an ==image search== engine.
